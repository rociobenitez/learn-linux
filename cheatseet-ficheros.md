# Directorios y Ficheros en Linux

| Directorio     | Fichero               | Descripción                                                                                                                                                                                  |
| -------------- | --------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `/bin`         |                       | _(enlace simbólico a `/usr/bin`)_                                                                                                                                                            |
| `/boot`        |                       | Archivos necesarios para el **arranque** del sistema, incluyendo el kernel de Linux (`vmlinuz`) y el gestor de arranque (GRUB)                                                               |
| `/dev`         |                       | Ficheros de **dispositivos**                                                                                                                                                                 |
| `/dev/console` |                       | Archivo de configuración de la consola del sistema                                                                                                                                           |
| `/dev/hd`      |                       | Representa discos duros IDE (antiguos)                                                                                                                                                       |
| `/dev/null`    |                       | Dispositivo especial que descarta toda la información escrita en él (similar a un "agujero negro")                                                                                           |
| `/dev/sda`     |                       | Representa el primer disco duro SATA/IDE del sistema                                                                                                                                         |
| `/dev/tty`     |                       | Consolas virtuales, el sistema simula la conexión a varios terminales de texto                                                                                                               |
| `/etc`         |                       | Fciheros de **configuración del sistema**, scripts de arranque, etc. _MUY IMPORTANTE_                                                                                                        |
|                | `/sudoers`            | Archivo de configuración que define qué usuarios tienen permisos para usar `sudo` y qué comandos pueden ejecutar                                                                             |
|                | `/passwd`             | Información básica de los **usuarios** del sistema                                                                                                                                           |
|                | `/group`              | Información de los **grupos** de usuarios del sistema                                                                                                                                        |
|                | `/shadow`             | **Contraseñas cifradas** de los **usuarios**                                                                                                                                                 |
|                | `/gshadow`            | **Contraseñas** de los **grupos** y los administradores de cada grupo                                                                                                                        |
|                | `/hosts`              | Mapeo de nombres de host a direcciones IP (similar a un DNS local)                                                                                                                           |
|                | `/hostname`           | Nombre del host (nombre del equipo en la red)                                                                                                                                                |
|                | `/default/useradd`    | Valores por defecto usados cuando se crea un usuario nuevo con el comando `useradd`                                                                                                          |
|                | `/useradd.conf`       | Valores por defecto usados cuando se crea un usuario nuevo con el comando `adduser`                                                                                                          |
|                | `/apt/sources.list`   | Lista de **repositorios** de software para sistemas basados en Debian (como Ubuntu)                                                                                                          |
|                | `/bash.bashrc`        | Script de configuración que se ejecuta para todos los usuarios al iniciar una sesión interactiva de bash                                                                                     |
|                | `/profile`            | Script de configuración que se ejecuta para todos los usuarios al iniciar sesión                                                                                                             |
|                | `/resolv.conf`        | Configuración de los servidores DNS que el sistema utilizará para resolver nombres de dominio                                                                                                |
|                | `/fstab`              | Archivo de configuración que define cómo se montan los sistemas de archivos y las particiones del disco al arrancar el sistema                                                               |
|                | `/network/interfaces` | Configuración de las interfaces de red del sistema                                                                                                                                           |
| `/home`        |                       | Directorios personales de los usuarios. Cada usuario tiene su propio subdirectorio dentro de `/home`                                                                                         |
| `/lib`         |                       | _(enlace simbólico a `/usr/lib`)_                                                                                                                                                            |
| `/lost+found`  |                       | Almacena información y procesos que se estaban ejecutando cuando el sistema cae                                                                                                              |
| `/media`       |                       | Punto de montaje para dispositivos extraíbles como USBs y CDs. _Lo hace el sistema automáticamente (ej: USB)_                                                                                |
| `/mnt`         |                       | Punto de montaje temporal para sistemas de archivos montados **manualmente**                                                                                                                 |
| `/opt`         |                       | _Optional_. Aplicaciones y paquetes de software adicionales instalados manualmente (ej: Google Chrome, Skype)                                                                                |
| `/proc`        |                       | Sistema de archivos virtual **dinámico** que proporciona información sobre los procesos en ejecución y el estado del sistema. No existe realmente en el disco, se monta desde la memoria RAM |
|                | `/meminfo`            | Información sobre el uso de la memoria del sistema                                                                                                                                           |
|                | `/cpuinfo`            | Información sobre la CPU del sistema                                                                                                                                                         |
|                | `/uptime`             | Tiempo que el sistema ha estado en funcionamiento desde el último arranque                                                                                                                   |
| `/root`        |                       | Directorio personal del usuario `root` (administrador del sistema)                                                                                                                           |
| `/sys`         |                       | _System_. Sistema de archivos **virtual** que proporciona información sobre el hardware y los dispositivos del sistema                                                                       |
| `/tmp`         |                       | Ficheros **temporales** del sistema                                                                                                                                                          |
| `/usr`         |                       | Aplicaciones y archivos compartidos por todos los usuarios del sistema. _Software del sistema_                                                                                               |
| `/usr/bin`     |                       | Aplicaciones **binarias o ejecutables** (**comandos esenciales** del sistema disponibles para todos los usuarios como `ls`, `cp`, `mv`, etc.)                                                |
| `/usr/sbin`    |                       | **Comandos de administración** del sistema, únicamente accesibles por superusuario (`root`)                                                                                                  |
| `/usr/lib`     |                       | Bibliotecas compartidas necesarias para ejecutar aplicaciones en `/usr/bin` y `/usr/sbin`                                                                                                    |
| `/usr/share`   |                       | Archivos compartidos entre aplicaciones, como documentación, iconos y otros recursos                                                                                                         |
| `/sbin`        |                       | _(enlace simbólico a `/usr/sbin`)_                                                                                                                                                           |
| `/var`         |                       | Archivos **variables** que cambian con el tiempo, como logs, colas de impresión y archivos temporales                                                                                        |
| `/var/log`     | `syslog`              | Archivo de registro que contiene mensajes del sistema y de diversas aplicaciones                                                                                                             |
|                | `auth.log`            | Archivo de registro que contiene eventos relacionados con la **autenticación** y la seguridad del sistema                                                                                    |
